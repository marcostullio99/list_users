# CRUD de Tarefas com Django

Base inicial de um projeto Django com um app `tarefas`, usando SQLite e estrutura simples para iniciantes.

## O que ja vem pronto

- Projeto principal configurado
- App `tarefas`
- Modelo inicial de tarefa
- CRUD basico com templates
- SQLite configurado por padrao

## Como rodar

1. Crie e ative um ambiente virtual.
2. Instale as dependencias com `pip install -r requirements.txt`.
3. Execute `py manage.py makemigrations`.
4. Execute `py manage.py migrate`.
5. Inicie o servidor com `py manage.py runserver`.

Abra `http://127.0.0.1:8000/`.
